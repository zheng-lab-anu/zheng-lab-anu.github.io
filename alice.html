Welcome to the Alice Synthetic 3D World

Alice is a synthetic 3D world. We built it to develop several new challenging computer vision benchmarks including tasks: person re-identification, car re-identification, scene semantic segmentaion and object detection. To create these benchmarks, we collected and designed a large number of 3D models in Alice and developed a highly flexible and controllable data synthesizing system based on Unity3D engin. There are more than 1200 identities and 1400 cars models, as well as thousands of scenes in Alice. Based on these 3D models, we have build three benchmars as follows:

  • PersonX
  • VehicleX
  • SceneX

New benchmarks for more tasks is coming!
